KPI Performance Dashboard

Power BI · DAX · Power Query 2026

Overview

An interactive executive dashboard built in Power BI to track key business KPIs including revenue, growth rate, and regional performance. Designed for weekly leadership reviews with dynamic slicers and drill-through capabilities.

Methodology

  • Connected Power BI to multiple data sources via Power Query — ERP exports and SQL database
  • Modelled a star schema with fact and dimension tables for optimal query performance
  • Wrote DAX measures for YoY growth, running totals, and rolling averages
  • Designed the report layout with slicers (region, product line, time period) and drill-through pages
